Consola de comando del sistema operativo
Git bash
Comandos de git:
git version --- verificar la version de git instalada y que efectivamente esta instalado
git init --- para iniciar un repositorio
git status --- sirve para verificar el estado del repositorio
     Rama actual
     La confirmacion actual
     el estado de los archivos --- untracked tracked
mv -- cambiar nombres de archivos y/o carpetas desde la consola
git add "nombreArchivo" -- indica a git que el archivo llamado "nombreArchivo" debe tener seguimiento (tracked)
git add . -- indica a git que todos los archivos que se encuentren "untracked" pasen a ser "tracked"
git commit -- ejecuta la confirmacion (commit)
git commit -m "mensaje" -- ejecuta la confirmacion con un mensaje corto
